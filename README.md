# ⚔️ Кибер-Арена

Сервис бронирования зон кибер-арены колледжа IT.МОСКВА с AI-ассистентом на GigaChat.

## Как запустить

### 1. Установить Node.js
Скачать: https://nodejs.org (кнопка LTS)

### 2. Установить зависимости
```
npm install
```

### 3. Настроить GigaChat API (для AI-ассистента)
```
cp .env.example .env
```
Открыть `.env` и вставить ключ:
```
GIGACHAT_AUTH_KEY=ваш_ключ_сюда
```

**Получить ключ:**
1. Зайти на https://developers.sber.ru/portal/products/gigachat-api
2. Зарегистрироваться → создать проект → выбрать тариф **Personal (бесплатно)**
3. Скопировать **Authorization Data** (строка в Base64)

### 4. Запустить
```
npm run dev
```
Откроется: http://localhost:5173

> Команда запускает одновременно прокси-сервер GigaChat (порт 3001) и Vite (порт 5173).

## Что реализовано

### Обязательный функционал (по ТЗ)
- ✅ Авторизация, регистрация, выход
- ✅ Профиль: уровень, XP, монеты, достижения, статистика
- ✅ Три зоны: игровая, тренировочная, командная
- ✅ Бронирование (выбор даты, времени, длительности)
- ✅ Отмена бронирования
- ✅ Запрет пересекающихся броней
- ✅ **Запрет бронирования во время учебных пар** (Пн–Пт, 9:00–17:00)
- ✅ Календарь занятости зон
- ✅ Все 5 расширяющих модулей: достижения, турниры, команды, лента, экономика
- ✅ Панель администратора (зоны, расписание пар, брони, турниры, пользователи, статистика)

### Сверх ТЗ
- 🤖 **AI-ассистент бронирования** на GigaChat — пишешь «хочу поиграть в пятницу вечером 3 часа», ИИ подбирает слот и предлагает подтвердить одной кнопкой
- 💾 **Сохранение данных** — всё хранится в localStorage, данные не теряются при перезагрузке
- ⚙️ **Настройка расписания пар** из админки — можно менять дни и часы

## Технологии
- React 18 + Vite
- Express.js (прокси GigaChat)
- GigaChat API (Сбер)
- localStorage для персистентности
- CSS без внешних UI-библиотек
